# Development

## Tools

- Use `bin/generate-index` to produce this README from entries in the repository.

## Notes

- When adding a new entry, commit using `bin/commit`
  - This uses the format 'Add "Title of entry" TIL'.
  - It re-generates the index and includes it as part of the commit.

## Styleguide

- Put command names and invocations in backticks: `git`, `git commit -a`
- Put keyboard shortcuts in backticks, use cmd for macOS "command": `cmd + shift + 3`
